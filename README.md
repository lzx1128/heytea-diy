## 📋 详细部署指南

### Vercel

1. Fork 或克隆本仓库到您的 GitHub/GitLab 账户
2. 登录 [Vercel](https://vercel.com/)，点击"New Project"
3. 导入您的仓库，使用默认设置
4. 点击"Deploy"
5. 出现预览画面，点击即可使用，可保存网址，方便下次使用
